# Tabular Insight Workbench

Aplicación web en Streamlit para cargar datasets tabulares, perfilar calidad de datos, aplicar limpieza auditable, ejecutar análisis exploratorio y generar reportes ejecutivos.

## Stack

- Python 3.12.2
- Streamlit
- pandas
- NumPy
- matplotlib
- scikit-learn
- seaborn
- pytest

## Ejecución local

```powershell
.\.venv\Scripts\activate
streamlit run app.py
```
